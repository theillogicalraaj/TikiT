## 🧠 Problem Statement

This project simulates strategic decision-making systems where actions affect multiple entities.
It models real-world scenarios like resource allocation and competitive planning.

---

## 💡 Solution

An interactive web-based strategy game where players:

* Control a shared stack
* Make tactical moves
* Predict outcomes
* Maximize final score

---

## 🎮 Features

* Real-time gameplay
* Stack-based strategy
* Turn-based logic
* Score calculation system
* Smooth UI & animations
